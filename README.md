# Disaster Alert and Response System (D.A.R.S)

This repository contains the source code for **D.A.R.S (Disaster Alert and Response System)**. Created as a project for COMP 3606, D.A.R.S is designed to assist in disaster management using alert and response mechanisms.

## Overview

**D.A.R.S** aims to provide a reliable platform for managing disaster alerts and coordinating response efforts. The system allows for issuing alerts, monitoring their status, and providing relevant information to all concerned authorities and individuals.

## Features

- Issue disaster alerts
- Manage and track active alerts
- Coordinate response teams
- Notification system for stakeholders
- Simple and extendable design

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/MikelMaharaj/D.A.R.S.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd D.A.R.S
   ```
3. **Build the project:**
   
   This project uses Java. You can use your preferred IDE (e.g., IntelliJ, Eclipse) or build from the command line using:

   ```bash
   javac -d bin src/**/*.java
   ```

## Usage

Run the application using:

```bash
java -cp bin <MainClass>
```
Replace `<MainClass>` with the main entry class name.

Please refer to the source code and comments for additional details on usage and configuration.
